安装 GitBook

安装 GitBook 的最好办法是通过 npm 。在终端提示符下，只需运行以下命令即可安装 GitBook 。

npm install gitbook-cli -g

执行下面的命令，查看 GitBook 版本，以验证安装成功。

gitbook -V
