chapter-fold

在 book.json 中添加下面的内容。

{
    "plugins": ["chapter-fold"]
}

然后执行下面的命令，安装插件依赖。

npm install --save gitbook-plugin-chapter-fold

安装完成后即可实现侧边栏目录折叠效果。
