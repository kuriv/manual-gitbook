# 配置文件

GitBook 允许使用灵活的配置自定义书籍，这些选项在 `book.json` 文件中指定。

```json
{
    "root": "",
    "structure": {
        "readme": "README.md",
        "summary": "SUMMARY.md"
    },
    "title": "GitBook",
    "description": "GitBook 教程",
    "author": "Kuriv Vesti",
    "isbn": "",
    "language": "zh-hans",
    "direction": "ltr",
    "gitbook": ">=3.0.0"
}
```
